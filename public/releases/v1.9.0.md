# Version 1.9.0

## DealCloud CRM Integration

Connect your workflows directly to DealCloud CRM! This release introduces seamless integration with DealCloud, enabling Private Equity, Real Estate, and Investment Banking firms to automate deal management, company tracking, and relationship intelligence workflows.

### âœ¨ New Features

- **DealCloud Integration**: Connect your Lunnoa Automate workflows to DealCloud CRM using API key authentication. Manage deals, companies, contacts, and all entry types directly from your automated workflows.

- **Five Core Actions**: Complete control over your DealCloud data:
  - **List Entries**: Query and search for deals, companies, contacts, or any entry type with filters, pagination, and saved views
  - **Get Entry**: Retrieve detailed information about specific entries by ID
  - **Create or Update Entries**: Automatically sync data from external systems into DealCloud with upsert capabilities
  - **Get Schema**: Discover available fields and entry types before creating or updating records
  - **Delete Entry**: Remove entries with explicit confirmation to prevent accidental deletions

- **Flexible Entry Management**: Support for all DealCloud entry types including deals, companies, contacts, and custom types. Use lookup columns for smart upsert logic or direct entry IDs for precise updates.
